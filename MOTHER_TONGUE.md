## **III. THE GRAMMAR (PHYSICS ENGINE)**

1. **Vector Logic**
   Reading order is `Subject → Operator → Object`
   `人 動 ⇒ 來`

2. **Scoping**
   `【 … 】` defines a contained system
   `【 心 ∈ 空 】`

3. **Recursion**
   `⭮` applies to the preceding object or scope
   `破 ⭮`

4. **Negation**
   `不` modifies the following glyph
   `絕` is active severance
   `不 善` / `絕 物`
